## Project Overview
This database system tracks and analyzes the prices of 51 essential commodities (Sensitive Price Indicator - SPI) across 17 major cities in Pakistan.

The data is sourced from the **Pakistan Bureau of Statistics (PBS)**.

## Key Features
* **Price Comparison:** Compare item prices between any two cities (e.g., Lahore vs. Karachi).
* **Inflation History:** View price trends for a single item over time.
* **Technology Stack:** SQL (MySQL/SQLite), Python.
